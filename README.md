# Selenium TestNG Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Features
- Page Object Model (POM)
- TestNG Annotations
- Screenshot on Failure
- Reusable utilities

## Project Structure
- src/test/java → Test classes
- src/main/java → Page classes
- utils → reusable methods

## How to Run
mvn test
